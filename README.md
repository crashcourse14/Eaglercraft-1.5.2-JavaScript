# Eaglercraft-1.5.2-JavaScript
This is a 100% port of Eaglercraft 1.5.2, but it's written in JavaScript. Not Java. You're welcome.


# Notice
This project used AI to translate Java to JavaScript. This project is a port, and is not used a personal project. Use this project how ever you like. The AI we're using is Claude Sonnet 4.6. 

# Why We're Doing This

Why does this project exist? It's to make so people who don't have a Java IDE (Eclipse, IntelliJ, etc...) can run, and modify Eaglercraft 1.5.2. Not only does it make it easer to run, mods, you don't have to mess with gradle errors, nor desktop runtime. Especially for people's OS that doesn't support/can't use IDEs as I listed above.
